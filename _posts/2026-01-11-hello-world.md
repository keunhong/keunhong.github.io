Python code with syntax highlighting:

```python
import numpy as np

def gaussian(x, mu=0, sigma=1):
    """Compute the Gaussian/Normal distribution."""
    return (1 / (sigma * np.sqrt(2 * np.pi))) * \
           np.exp(-0.5 * ((x - mu) / sigma) ** 2)
```
